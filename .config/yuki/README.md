# Paquetes a instalar

Los paquetes que necesito:

- AUR (yay):
```
git clone https://aur.archlinux.org/yay-bin.git
cd yay-bin
makepkg -si
```

- base:
  - neovim terminator yadm zsh etckeeper podman podma-compose  podlet qemu-full virt-manager nvidia-settings

- hyprland y windowmanager:
  - mako waybar hyprpaper hyprpolkitagent ttf-nerd-fonts-symbols ttf-nerd-fonts-symbols-mono ttf-sourcecodepro-nerd satty hyprshot
